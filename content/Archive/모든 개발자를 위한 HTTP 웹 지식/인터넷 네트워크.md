# IP
## 인터넷 프로토콜 역할
* 지정한 IP 에 데이터 전달
* 패킷  이라는 통신 단위로 데이터 전달
	* ip패킷 정보 
		* 출발지 IP , 목적지ip,기타
## IP 프로토콜의 한계
* 비 연결성
	* 패킷을 받을 대상이 없거나 서비스 불능 상태여도 패킷 전송
* 비 신뢰성
	* 중간에 패킷 이 사라질 경우
	* 패킷 이 순서대로 안 올 경우
* 프로그램 구분
	* 같은 IP 를 사용하는 서버에서 통신하는 애플리케이션이 둘 이상이면?
